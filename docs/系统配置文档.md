| JDK | 17 | 21 LTS |

| Redis | 7.0 | 7.2 (Redis Stack) |

| Node.js | 18 LTS | 20 LTS |
| ImageMagick | 7.0 | 7.1 |

---

## 环境配置详解（.env）

`.env` 文件位于项目根目录，是所有配置的唯一入口。

```env
# ---- MySQL 数据库 ----
DB_HOST=localhost
DB_PORT=3306
DB_NAME=diagrammnt
DB_USERNAME=root
DB_PASSWORD=your_db_password           # 请修改为强密码

# ---- Redis ----
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=your_redis_password     # 无密码则留空
REDIS_DATABASE=0

# ---- SSO 统一登录 ----
SSO_JWT_SECRET=change-me-to-a-real-secret-at-least-256-bits

# ---- NAS 文件存储 ----
NAS_MOUNT_PATH=/mnt/nas/product-images

# ---- 特征提取 ----
FEATURE_MODEL_PATH=./models/resnet50.onnx
FEATURE_EXTRACT_WORKERS=4
FEATURE_BATCH_SIZE=100

# ---- 图片处理 ----
IMAGE_MAX_UPLOAD_SIZE_MB=50
IMAGE_ALLOWED_FORMATS=jpg,jpeg,png,psd,webp,bmp,gif

# ---- 日志 ----
LOG_LEVEL=INFO
LOG_PATH=./logs
```

## 数据库初始化

```sql
CREATE USER 'diagrammnt'@'localhost' IDENTIFIED BY 'your_password';
CREATE DATABASE diagrammnt DEFAULT CHARACTER SET utf8mb4;
GRANT ALL PRIVILEGES ON diagrammnt.* TO 'diagrammnt'@'localhost';
```

## 构建部署

### 后端
```bash
cd backend
mvn clean package -DskipTests
java -jar target/diagrammnt-backend-1.0.0-SNAPSHOT.jar
```

### 前端
```bash
cd frontend
npm install
npm run build
```

生产构建后将 `frontend/dist/` 复制到 `backend/src/main/resources/static/`，重新打包即可单 JAR 部署。
